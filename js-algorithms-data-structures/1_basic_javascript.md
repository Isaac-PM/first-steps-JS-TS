# Basics

## Print to the console

```javascript
console.log("Hello World!");
```

## Variables

Variables are declared using the `let` keyword, the default naming convention is camelCase.

```javascript
let myName = "John Doe";
```

### Data types

## Constants

Constants are declared using the `const` keyword.

```javascript
const MY_NAME = "John Doe";
```
